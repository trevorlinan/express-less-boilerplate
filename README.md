# Express-Less Boilerplate

## Installation

    > git clone https://github.com/trevorlinan/express-less-boilerplate.git <Your App Name>
    > cd <Your App Name>
    > npm install
    
## Start Application

    > npm run dev
    
#### Manually run LESS Compiler and Minify CSS

    > npm run less
    
#### Get Latest Updates

    > git pull
