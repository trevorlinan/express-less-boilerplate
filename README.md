# Express-Less Boilerplate

## Installation

    > Fork this repo
    > git clone https://github.com/<Your URL>/express-less-boilerplate.git <Your App Name>
    > cd <Your App Name>
    > npm install
    
## Start Application

    > npm run dev
    
#### Manually run LESS Compiler and Minify CSS

    > npm run less
    
#### Get Latest Updates

    > git pull
